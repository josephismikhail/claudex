## The default workflow

```powershell
claudex
```

Claudex opens its own **Joey's Claudex** terminal while retaining the installed
Claude agent harness underneath. A brand-new installation shows `/model to
authenticate`. Run `/model`, choose a provider, finish authentication in the
browser, and select a returned model. The picker updates without restarting or
choosing a profile.

## Supported account flows

| Provider | Authentication | Catalog |
|---|---|---|
| OpenAI | ChatGPT browser OAuth | Picker-visible models returned for that account |
| Anthropic API | Console API key | Models returned for that key by Anthropic's Models API |

Anthropic prohibits third-party gateways from routing Claude Free, Pro, or Max
credentials, so Claudex intentionally uses Console API keys for Anthropic.

Advanced legacy profiles remain available for local models and custom
OpenAI-compatible endpoints, but new users do not need to configure them.
